# 🛡️ بوت الحماية (Security Bot) لتيليجرام

بوت تيليجرام لفحص الروابط والملفات من الفايروسات، ومعرفة معلومات عناوين IP،
مع لوحة تحكم للأدمن واسترجاع كلمات سر ضعيفة لملفات شخصية.

## ⚙️ المميزات

- 🔗 فحص الروابط من الفايروسات والمواقع الضارة (عبر VirusTotal)
- 📎 فحص الملفات بعشرات محركات الحماية (عبر VirusTotal)
- 🌍 معرفة الدولة والمدينة ومزود الخدمة لعنوان IP (عبر ip-api.com)
- 📄 تقرير PDF تلقائي بعد كل فحص
- 🔑 استرجاع كلمة سر ضعيفة/شائعة لملفات ZIP/RAR/PDF شخصية
- 🛠️ لوحة تحكم أدمن كاملة داخل تيليجرام (إحصائيات، حظر، بث رسائل)
- 💾 قاعدة بيانات SQLite لتخزين المستخدمين وسجل الفحوصات

## 📋 المتطلبات قبل التشغيل

1. **Python 3.10+**
2. **توكن بوت تيليجرام**: احصل عليه من [@BotFather](https://t.me/BotFather)
3. **مفتاح VirusTotal API** (مجاني): سجّل حساب على
   https://www.virustotal.com/gui/join-us ثم اذهب لإعدادات حسابك
   وانسخ API Key

   ⚠️ ملاحظة: الحساب المجاني محدود بحوالي 4 طلبات/دقيقة و500 طلب/يوم،
   كافٍ للاستخدام الشخصي أو مجتمع صغير. لحجم استخدام أكبر تحتاج خطة مدفوعة.

## 🚀 التثبيت

```bash
# 1. تثبيت المكتبات
pip install -r requirements.txt

# 2. (اختياري) لدعم ملفات RAR، فعّل السطر في requirements.txt
#    وثبّت أداة unrar على نظامك:
#    Ubuntu/Debian: sudo apt install unrar
#    Windows: نزّل WinRAR أو UnRAR من الموقع الرسمي

# 3. ضبط التوكنات - الطريقة الموصى بها (متغيرات بيئة):
export TELEGRAM_BOT_TOKEN="التوكن_هنا"
export VIRUSTOTAL_API_KEY="المفتاح_هنا"

# أو عدّل القيم مباشرة في ملف config.py (أقل أماناً، لا ترفعه على GitHub لو فعلت هذا)
```

## 👤 ضبط الأدمن

افتح `config.py` وعدّل:

```python
ADMIN_IDS = [123456789]  # ضع آيدي تيليجرام الخاص بك
```

لمعرفة آيديك على تيليجرام: راسل [@userinfobot](https://t.me/userinfobot)

## ▶️ التشغيل

```bash
python bot.py
```

سترى رسالة `🚀 البوت بدأ التشغيل...` في الطرفية، والبوت جاهز الآن على تيليجرام.

## 🛠️ أوامر الأدمن (داخل تيليجرام)

| الأمر | الوظيفة |
|---|---|
| `/admin` | عرض لوحة التحكم وقائمة الأوامر |
| `/admin_stats` | إحصائيات عامة (عدد المستخدمين، الفحوصات، التهديدات) |
| `/admin_users` | عرض آخر 20 مستخدم |
| `/admin_ban <user_id>` | حظر مستخدم |
| `/admin_unban <user_id>` | فك حظر مستخدم |
| `/admin_scan <scan_id>` | تفاصيل فحص معيّن |
| `/admin_broadcast <رسالة>` | إرسال رسالة لجميع المستخدمين |

## 👥 أوامر المستخدم العادي

| الأمر | الوظيفة |
|---|---|
| `/start` | بدء استخدام البوت |
| `/help` | عرض المساعدة |
| `/history` | سجل آخر 10 عمليات فحص |
| `/recover_password` | محاولة استرجاع كلمة سر لملف شخصي |
| إرسال رابط مباشرة | فحص الرابط تلقائياً |
| إرسال IP مباشرة | فحص عنوان IP تلقائياً |
| إرسال ملف مباشرة | فحص الملف تلقائياً |

## ⚠️ حدود مهمة يجب معرفتها

1. **استرجاع كلمة السر**: يعمل فقط مع كلمات سر ضعيفة/شائعة أو قريبة من
   تلميحات تزوّدها. هذا **ليس خللاً** بل هو الهدف من التشفير الجيد:
   كلمة سر قوية وعشوائية لن تُكتشف، ولا توجد طريقة تقنية لـ"فك تشفير"
   AES أو تشفير حديث آخر بدون المفتاح/كلمة السر الصحيحة، بغض النظر عن
   عدد "الطبقات" المفترضة.

2. **حصة VirusTotal المجانية محدودة**: لو واجهت أخطاء "rate limit"،
   هذا يعني تجاوز الحصة اليومية/الدقيقية. فكّر بترقية الخطة لو الاستخدام كبير.

3. **استخدام مسؤول**: هذا البوت مصمم لمساعدتك في فحص ما تملكه أو تشك فيه.
   لا تستخدمه لمحاولة الوصول لملفات أو حسابات لا تملكها.
